Introduction to NLP

The Core Idea

Natural Language Processing (NLP) is a branch of artificial intelligence that helps computers understand, interpret, and manipulate human language. It bridges the gap between human communication and computer understanding.

The Language Detective

Think of NLP like a detective analyzing a conversation. Just as a detective looks for clues in speech patterns, word choice, and context to understand meaning, NLP systems analyze text by breaking it down into meaningful components and examining their relationships.

  • Words are like individual clues that contribute to meaning
  • Context is like the crime scene - it gives clues their significance
  • Grammar rules are like investigative procedures
  • Understanding intent is like solving the case

Technical Implementation

import nltk
from nltk.tokenize import word_tokenize
from nltk.corpus import stopwords

def basic_text_analysis(text):
    # Tokenize the text into words
    tokens = word_tokenize(text.lower())
    
    # Remove stopwords
    stop_words = set(stopwords.words('english'))
    meaningful_words = [w for w in tokens if w not in stop_words]
    
    # Get word frequencies
    freq_dist = nltk.FreqDist(meaningful_words)
    
    return {
        'token_count': len(tokens),
        'unique_words': len(set(meaningful_words)),
        'most_common': freq_dist.most_common(5)
    }

# Example usage
text = "Natural language processing helps computers understand human language."
analysis = basic_text_analysis(text)
print(analysis)

Text Preprocessing

The Core Idea

Text preprocessing is the crucial first step in NLP that transforms raw text into a clean, standardized format that machines can process effectively. This includes tasks like tokenization, removing stop words, and normalizing text.

The Text Kitchen

Think of text preprocessing like preparing ingredients in a kitchen. Just as a chef needs to wash, peel, and chop ingredients before cooking, NLP systems need to clean and prepare text data before analysis. Different recipes (NLP tasks) require different types of preparation.

  • Raw text is like unprocessed ingredients
  • Tokenization is like chopping ingredients into pieces
  • Removing stop words is like removing unwanted parts
  • Text normalization is like standardizing ingredient sizes

Technical Implementation

import re
from nltk.tokenize import word_tokenize
from nltk.corpus import stopwords
from nltk.stem import WordNetLemmatizer

def preprocess_text(text):
    # Convert to lowercase
    text = text.lower()
    
    # Remove special characters and numbers
    text = re.sub(r'[^a-zA-Z\s]', '', text)
    
    # Tokenize
    tokens = word_tokenize(text)
    
    # Remove stopwords
    stop_words = set(stopwords.words('english'))
    tokens = [t for t in tokens if t not in stop_words]
    
    # Lemmatize
    lemmatizer = WordNetLemmatizer()
    tokens = [lemmatizer.lemmatize(t) for t in tokens]
    
    return tokens

# Example usage
text = "The quick brown foxes are jumping over the lazy dogs!"
clean_tokens = preprocess_text(text)
print(clean_tokens)

Tokenization and Vocabulary

The Core Idea

Tokenization is the process of breaking text into smaller units (tokens) such as words or subwords. Building a vocabulary from these tokens is essential for converting text into a format that machine learning models can process.

The Language Building Blocks

Think of tokenization like breaking down a LEGO structure into its individual blocks. Just as LEGO blocks are the basic units for building larger structures, tokens are the fundamental units for understanding text. The vocabulary is like your collection of unique LEGO pieces that you can use to build any structure.

  • Words are like individual LEGO blocks
  • Sentences are like assembled structures
  • Vocabulary is like your LEGO collection catalog
  • Word relationships are like block connection patterns

Technical Implementation

from collections import Counter
from nltk.tokenize import word_tokenize, sent_tokenize

class VocabularyBuilder:
    def __init__(self, min_freq=2):
        self.word2idx = {}
        self.idx2word = {}
        self.word_freq = Counter()
        self.min_freq = min_freq
    
    def build_vocab(self, texts):
        # Count word frequencies
        for text in texts:
            tokens = word_tokenize(text.lower())
            self.word_freq.update(tokens)
        
        # Create vocabulary (only include words above minimum frequency)
        vocab = [word for word, freq in self.word_freq.items() 
                if freq >= self.min_freq]
        
        # Create mappings
        for idx, word in enumerate(vocab):
            self.word2idx[word] = idx
            self.idx2word[idx] = word
    
    def encode(self, text):
        tokens = word_tokenize(text.lower())
        return [self.word2idx.get(token, -1) for token in tokens]

# Example usage
texts = [
    "Natural language processing is fascinating.",
    "Processing text requires careful analysis."
]

vocab_builder = VocabularyBuilder(min_freq=1)
vocab_builder.build_vocab(texts)
encoded = vocab_builder.encode(texts[0])
print(f"Vocabulary size: {len(vocab_builder.word2idx)}")
print(f"Encoded text: {encoded}")
